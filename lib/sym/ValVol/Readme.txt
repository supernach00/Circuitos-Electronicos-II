My library models for the simulator LTspice. The archive contains the model:
1. PWM controller TL494.
2. PWM controller UC3825
3. PWM controller SG3525A
4. Phase Shift Resonant Controller UC3875
5. Phase Shift Resonant Controller UC3879
6. Flyback Controller UCC28610
7. PFC controller UC2854
8. PFC controller UCC28070
9. Driver Optocoupler HCPL3180
10. Optocoupler driver HCPL316
11. Triac driver optocoupler MOC3082
12. Triac driver optocoupler MOC3052
13. The model of the operational amplifier K544UD2 (borrowed from the library r-opamp.lib Micro-Cap)
14. G_loop module for measuring the hysteresis loop of a nonlinear inductance
15. Winding of ideal transformer

Unzip the archive to a folder sym / ValVol.
